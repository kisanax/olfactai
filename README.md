# 👃 OlfactAI — AI Smell Intelligence Agent

> *"Every sense is now digital — except smell. OlfactAI is the missing bridge."*

Built for **AMD Developer Hackathon: ACT II** 🏆

## 🌍 The Problem

- 🇮🇩 Indonesia has millions of LPG gas users — gas leaks kill hundreds yearly
- 1 in 8 COVID survivors lost their sense of smell permanently  
- Food poisoning still causes thousands of deaths in developing countries

**The nose is the last undigitized human sense. OlfactAI changes that.**

## 💡 What is OlfactAI?

OlfactAI is an AI-powered smell intelligence agent that:
- 🔍 **Classifies smells** from text or voice descriptions
- ⚠️ **Detects danger** — LPG gas leaks, smoke, rotten food, toxic chemicals
- 🚨 **Gives emergency instructions** in real-time (Bahasa Indonesia + English)
- 🏥 **Supports medical screening** for smell-related conditions (anosmia)

## 🎯 Demo Scenarios

| Input | Detection | Action |
|-------|-----------|--------|
| "Bau manis menyengat di dapur" | 🔴 DANGER — LPG Gas | Emergency evacuation steps |
| "Daging berbau asam aneh" | 🟡 WARNING — Rotten food | Do not consume |
| "Bau tanah setelah hujan" | 🟢 SAFE — Petrichor | Normal, enjoy nature |
| "Bau asap dari ruangan sebelah" | 🔴 DANGER — Fire/smoke | Call emergency services |

## 🛠️ Tech Stack

- **AI Engine:** Fireworks AI API (AMD-powered infrastructure)
- **Backend:** Python + Flask
- **Frontend:** HTML/CSS/JavaScript
- **Compute:** AMD Developer Cloud

## 🚀 Quick Start

```bash
git clone https://github.com/kisanax/olfactai.git
cd olfactai
pip install -r requirements.txt
echo FIREWORKS_API_KEY=your_key_here > .env
python app.py
```
